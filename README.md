## General info
This project is a basic implemention of a well-know game Rock Paper Scissors, done as an assignment from The Odin Project.
The purpose of this project, was to test my skills of pure JavaScript and DOM manipulation.
Through making this simple game I got hands on experience with the DOM and I've also gotten better at problem solving with JavaScript.
